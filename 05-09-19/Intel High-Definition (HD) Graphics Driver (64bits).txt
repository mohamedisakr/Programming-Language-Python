TITLE:  Intel High-Definition (HD) Graphics Driver (64bits)

VERSION:  8.15.10.2430 REV:  A
DESCRIPTION:  
This package provides the driver that enables the Intel High-Definition (HD) Graphics in supported notebook models that are running a supported operating system. If high-definition multimedia interface (HDMI) video output is supported, this package also installs the HDMI Audio Driver.

PURPOSE:  Recommended
SoftPaq NUMBER:  SP54976
EFFECTIVE DATE:  October 25, 2011
CATEGORY:  Driver - Graphics
SSM SUPPORTED:  YES

DEVICES SUPPORTED:  
PCI\VEN_8086&DEV_0126="Intel(R) HD Graphics Family"
PCI\VEN_8086&DEV_0046="Intel(R) HD Graphics"


PREREQUISITES:  
The Intel HM65 or HM55 Express Chipset is required. NOTE: To determine the installed chipset, look in the Microsoft Windows Device Manager under System Devices.

FIXES:  
Fixed some characters show incorrectly in Singapore localization.

HOW TO USE:  
1. Download the file by clicking the "Download" or "Obtain Software" button and saving the file to a folder on your hard drive (make a note of the folder where the downloaded file is saved).

2. Double-click the downloaded file and follow the on-screen instructions.
